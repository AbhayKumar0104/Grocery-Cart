# 🛒 Grocery Cart - Full Stack MERN Application

A full-stack **Grocery Cart e-commerce web application** built using the MERN stack.  
It allows users to browse products, manage a shopping cart, place orders, and manage authentication — all with a modern and responsive UI.

## 📌 Features

### 👤 User Features
- User registration and login (JWT authentication)
- Browse grocery products
- Add / remove items from cart
- Update product quantity
- Place orders
- View order history

### 🛍️ Seller Features
- Seller authentication
- Add new products
- Manage product listings
- View customer orders

### ⚙️ System Features
- RESTful API architecture
- Secure authentication using JWT
- MongoDB Atlas integration
- Image upload using Cloudinary
- Middleware-based route protection

---

## 🧠 Tech Stack

### Frontend
- React.js

- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
